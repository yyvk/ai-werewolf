## 💻 代码类（可直接运行）

### 1️⃣ werewolf_agent_demo.py ⭐ **零配置运行**
**内容：**
- 基础版Agent架构演示
- 完整的9人游戏流程
- 3种AI性格展示
- 纯Python实现，无需API

**特点：**
- ✅ 无需任何配置
- ✅ 立即可运行
- ✅ 理解Agent架构
- ✅ 适合快速验证

**运行：**
```bash
python werewolf_agent_demo.py
```

**代码量：** ~400行

---

### 2️⃣ werewolf_with_llm.py ⭐ **真实LLM版本**
**内容：**
- 接入OpenAI/通义千问等LLM
- 智能Agent实现
- Prompt Engineering示例
- 成本优化方案

**特点：**
- ✅ 真实的AI推理
- ✅ 高质量的发言
- ✅ 支持多种LLM
- ✅ 生产环境可用

**运行：**
```bash
# 1. 配置API密钥
export OPENAI_API_KEY=your-key

# 2. 运行演示
python werewolf_with_llm.py

# 3. 查看实施指南
python werewolf_with_llm.py guide
```

**代码量：** ~600行
