# ============================================================
# 常见问题解决
# ============================================================
#
# 1. 如果遇到 langchain 版本冲突：
#    pip install --upgrade langchain langchain-core langchain-community langchain-openai
#
# 2. 如果遇到 pydantic 版本冲突：
#    pip install --upgrade "pydantic>=2.0.0"
#
# 3. 如果 Windows 上 uvicorn 安装失败：
#    pip install uvicorn（不带[standard]）
#
# 4. 如果 dashscope 导入失败：
#    pip install --upgrade dashscope
