# 游戏对局界面优化说明

## 优化内容

### 1. 动态背景切换（黑夜/白天）
- **白天背景**: 采用蓝天绿地的渐变效果，配合动态太阳☀️图标
- **黑夜背景**: 采用深蓝色夜空渐变，配合星星闪烁效果和月亮🌙图标
- 背景会根据游戏阶段（phase）自动切换，转场流畅

### 2. 玩家圆形布局（村庄视图）
- 将玩家从网格布局改为圆形布局，更像参考图片的效果
- 每个玩家有：
  - **圆形头像**: 显示角色图标（狼人🐺、村民👨、预言家🔮、女巫🧙等）
  - **编号徽章**: 右上角显示玩家编号
  - **名牌**: 显示玩家名称和角色
  - **动画效果**: 
    - 存活玩家有脉冲动画
    - 发言玩家有缩放动画和光晕效果
    - 悬浮时放大并显示详细信息卡片

### 3. AI角色图标系统
根据角色显示不同的图标：
- 🐺 狼人
- 👨 村民
- 🔮 预言家
- 🧙 女巫
- 🏹 猎人
- 🛡️ 守卫
- 💀 已淘汰玩家
- 🤖 未分配角色

### 4. 中央舞台
- 在玩家圆圈中央显示当前发言玩家信息
- 带有角色图标和动态动画效果
- 没有发言时显示提示信息

### 5. 增强的界面元素
- **阶段指示器**: 在标题下方显示当前是黑夜还是白天，以及轮次
- **玩家悬浮提示**: 鼠标悬停在玩家上时显示详细信息卡片
- **流式发言显示**: 改进的发言显示，包含角色图标和打字动画
- **响应式设计**: 适配各种屏幕尺寸

### 6. 夜间模式适配
- 所有界面元素都针对黑夜模式进行了优化
- 卡片背景、文字颜色、边框都会根据阶段自动调整
- 保持良好的可读性和视觉效果

## 技术实现

### 新增函数
- `isNightPhase()`: 判断是否为夜晚阶段
- `getPhaseClass()`: 获取阶段样式类
- `getPhaseText()`: 获取阶段文本显示
- `getAllPlayers()`: 获取所有玩家（包括占位符）
- `getPlayerPosition()`: 计算玩家圆形布局位置
- `getPlayerIcon()`: 获取玩家图标
- `getRoleIcon()`: 获取角色图标

### 样式特性
- 使用 CSS 动画: `pulse`, `speaking`, `bounce`, `fadeIn`, `slideIn` 等
- 背景渐变和星星效果
- 半透明玻璃态效果（backdrop-filter）
- 平滑的过渡动画（transition）

## 使用效果

1. **沉浸式体验**: 黑夜和白天的背景切换增强了游戏氛围
2. **直观展示**: 圆形布局让玩家位置更清晰
3. **信息丰富**: 角色图标和悬浮提示提供更多信息
4. **视觉反馈**: 发言玩家、存活状态等都有明确的视觉反馈
5. **响应式**: 在不同设备上都能良好显示
