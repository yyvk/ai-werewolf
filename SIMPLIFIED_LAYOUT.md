# 简化玩家布局优化

## 🎯 优化目标

将玩家卡片简化为**只显示头像和昵称**，打造更简洁清爽的游戏界面。

## ✨ 优化内容

### 之前的布局
```
┌─────────────────┐
│ [编号徽章]      │
│   🐺 大头像      │
│   玩家昵称      │
│   角色标签      │
└─────────────────┘
```

### 优化后的布局
```
┌─────────────┐
│             │ [编号]
│   🐺        │
│   大头像     │
│             │
│  玩家昵称    │
└─────────────┘
```

## 📋 主要改进

### 1. **卡片简化**
- ✅ 移除了角色标签显示
- ✅ 移除了详细信息提示框
- ✅ 保留核心信息：头像 + 昵称
- ✅ 编号徽章移到右上角

### 2. **视觉优化**
- **头像尺寸**: 从 2.5rem 增大到 **3.5rem**
- **卡片尺寸**: 从 200px 缩小到 **140px**（更紧凑）
- **边框**: 3px 实线边框，状态不同颜色不同
- **间距**: 缩小到 0.8rem，排列更紧密

### 3. **状态指示**

#### 存活玩家
- 边框：绿色 (#4caf50)
- 动画：呼吸灯效果（aliveGlow）
- 头像：正常显示

#### 已淘汰玩家
- 边框：灰色 (#9e9e9e)
- 透明度：50%
- 滤镜：灰度 50%

#### 正在发言
- 边框：紫色 (#667eea)
- 动画：脉冲效果 + 发光阴影
- 头像：上下跳动 + 放大
- 缩放：1.05倍

#### 空槽位
- 头像：❓（问号）
- 边框：虚线
- 透明度：30%

### 4. **编号徽章设计**
- 位置：右上角 (-8px, -8px)
- 尺寸：32x32px
- 样式：渐变背景 + 白色边框
- 阴影：0 3px 10px

### 5. **响应式适配**

#### 大屏幕 (> 1400px)
- 卡片：140x140px
- 头像：3.5rem

#### 中等屏幕 (1200px - 1400px)
- 卡片：130x130px
- 头像：3rem

#### 移动端 (< 768px)
- 卡片：110x110px
- 头像：2.5rem
- 昵称：0.85rem

## 🎨 设计细节

### 玩家卡片
```css
.player-card-simple {
  background: rgba(255, 255, 255, 0.95);
  border: 3px solid transparent;
  border-radius: 15px;
  padding: 1rem;
  backdrop-filter: blur(10px);
}
```

### 头像动画
```css
@keyframes iconBounce {
  0%, 100% {
    transform: translateY(0) scale(1);
  }
  50% {
    transform: translateY(-8px) scale(1.1);
  }
}
```

### 呼吸灯效果
```css
@keyframes aliveGlow {
  0%, 100% {
    box-shadow: 0 3px 10px rgba(76, 175, 80, 0.3);
  }
  50% {
    box-shadow: 0 3px 20px rgba(76, 175, 80, 0.6);
  }
}
```
